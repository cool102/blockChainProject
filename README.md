# Blockchain Project (учебный)

Небольшой учебный проект на Java, посвящённый базовым принципам блокчейна: блоки, цепочка, хеширование, (опционально) proof-of-work и валидация целостности.

## Зачем проект
- Понять, как формируется блок и как связываются блоки в цепочку
- На практике увидеть, почему изменение данных “ломает” весь блокчейн
- Потрогать криптографическое хеширование и проверку целостности

## Возможности
- Создание блоков и добавление в цепочку
- Хеширование блока (обычно SHA-256)
- Проверка валидности цепочки
- (Опционально) Proof-of-Work / сложность майнинга: `difficulty`, `nonce`  *(TODO: если есть — опиши параметры)*
